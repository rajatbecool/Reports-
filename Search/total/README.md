# Total Search Evidence

This folder is generated from the 20 non-patent `Search/*/evidence.csv` workstream folders.

Files:
- `combined_evidence.csv`: all evidence rows combined, with source-folder traceability.
- `unique_links.csv`: de-duplicated exact URLs extracted from the combined source fields.
- `summary.csv`: row and link counts by workstream plus total.

Generation rule: the original 20 evidence folders remain the source universe. This folder is only an aggregate view.
